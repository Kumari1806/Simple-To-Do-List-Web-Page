# Simple-To-Do-List-Web-Page
Here is the summary of To-Do List web page:

A web page is made for a to-do list using html,css and javascript. 
In html, a container is made where it has one button called add task.
It will take the input from the user.
It will be added into the list and empty the placeholder.
As the task is added, the container box will be increased as per requirement.
Delete button is also provided in case the user wants to delete the written task.
To make it more appealing to the user, different colors has been used.
Hover is used when the cursor will point the button either add/delete button.
To understand how task are being added or deleted, javascript is used.
Here, addTask has been called and created a button called add task.
Same case is with the delete button also, deleteBtn has been used to delete the item from the input list.
Having a responsive website makes an user to use site as per his convenience.
This website is a responsive website, where an user can easily open the web page in laptop, tablet or phone.
